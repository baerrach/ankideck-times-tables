# Anki #

## Add-ons ##

The following add-ons are needed to make importing/exporting easier.

See [Add-ons in the Anki Documentation](https://docs.ankiweb.net/addons.html) for more information.

To install the add-ons use `Tools > Add-ons > Get Add-ons...`.
For each add-on enter the code listed below and click the `OK` button.

* 1672832404 - [Add note id](https://ankiweb.net/shared/info/1672832404)
* 46741504 - [Allows empty first field (during adding and import)](https://ankiweb.net/shared/info/46741504)
  (this appears to be out of date)
  guids can be generated by formula =CONCATENATE(DEC2HEX(RANDBETWEEN(0,4294967295),8),"-",DEC2HEX(RANDBETWEEN(0,65535),4),"-",DEC2HEX(RANDBETWEEN(0,65535),4),"-",DEC2HEX(RANDBETWEEN(0,65535),4),"-",DEC2HEX(RANDBETWEEN(0,4294967295),8),DEC2HEX(RANDBETWEEN(0,65535),4))
